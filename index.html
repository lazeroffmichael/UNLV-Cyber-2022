Welcome To The STEM Summer 2022 Cybersecurity Class!¶

This is an introductory class to the world of cybersecurity. We will be studying different topics ranging from Password Cracking, Networks, Internet Security, Physical Security and more. Hopefully by the end of this class you will pick up tools and strategies that will be useful in the future.
